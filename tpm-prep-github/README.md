# Principal TPM Interview Prep

Static HTML interview prep site for Principal TPM roles. No build step, no backend.

## 🚀 Deploy to GitHub Pages

1. Upload all files to a new repo
2. Settings → Pages → Source: main branch → / (root) → Save
3. Visit: `https://<username>.github.io/<repo>/`

## Features

- 📅 6-Week Schedule with curated videos, courses, hands-on labs
- 🧩 43 Design Patterns (System Design + AI/GenAI)
- 🎙️ Multi-turn AI Interviewer with follow-ups (Ollama-powered, free)
- 📄 Resume & Prep Agent with voice recording
- 📚 Online-only resources (no books — YouTube, Coursera, AWS, LinkedIn, free tools)
- ✅ Progress Tracker with localStorage

## AI Backend

Uses [Ollama](https://ollama.com/) (free, local). Install: `curl -fsSL https://ollama.com/install.sh | sh && ollama pull llama3.1`

Also works with copy-paste to any free AI (Gemini, HuggingChat, ChatGPT, Claude).
